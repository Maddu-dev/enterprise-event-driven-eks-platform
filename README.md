# Enterprise Event-Driven EKS Platform

## Overview
Production-grade, event-driven microservices platform built on AWS EKS using
Terraform, Kafka (MSK), Redis (ElastiCache), EFS, and KMS with GitOps delivery.

## Key Capabilities
- Event-driven microservices using Kafka (MSK)
- Secure AWS EKS platform with IRSA
- Terraform-based infrastructure (modular & multi-env)
- Centralized observability and logging
- Encryption at rest using KMS
- GitOps-based deployments with ArgoCD
